THIS IS REEL'S DATA FOLDER
==========================

The server saves everything that needs to persist in HERE, so the
rest of the app folder stays clean. Files that appear automatically:

  profiles.json     All profiles, the kids/admin flags, per-profile
                    watch history and "My List". PINs and security-
                    question answers are stored SALTED + HASHED here
                    (scrypt) — the actual codes are never written in
                    plain text.

  tmdb-cache.json   Cached artwork/info lookups (only if you added a
                    TMDB key). Safe to delete; it just rebuilds.

BACK UP YOUR SETUP
  To save or move your profiles + history to another PC, copy this
  whole "data" folder. Drop it next to server.js on the other machine
  and everything comes back.

RESET
  Delete profiles.json to wipe all profiles and history (a fresh
  Admin, password 3119, is re-created on next start).

FORGOT THE ADMIN PASSWORD
  Stop the server, open profiles.json, delete the entire "admin"
  profile object from the "profiles" list, save, restart — a new
  Admin (password 3119) is re-seeded.

Do not put movies or shows in here — those go in the movies\ and
series\ folders. You can delete this note file; the folder stays.
